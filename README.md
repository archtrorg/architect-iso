architect-iso
==================

### Source for Architect iso builds with
`cli_installer` [aif](https://github.com/archtrorg/aif)
`cli_installer-dev` [aif-dev](https://github.com/archtrorg/aif-dev)

### Build
git clone https://github.com/archtrorg/architect-iso

cd architect-iso

sudo ./build.sh -v
